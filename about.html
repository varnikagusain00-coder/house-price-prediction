About This Project

This website provides a simple tool to predict house prices based on user input features such as number of bedrooms, bathrooms, square footage, and location.

The prediction is done using a basic formula for demonstration purposes. The project is built using HTML, CSS, PHP, JavaScript, and MySQL database for data storage.
